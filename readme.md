### Основные функции

`sorted(a)` - сортировка

`s.count("A")` - количество символов

`range(10)` - генерация чисел

### Логические выражения 

Проверка, что строка входит в другую строку:

```python
a = "a"
print(a in "abc") # True
print(a not in "abc") # False
```

#### Алгебра-логика

`∨` - `or` или

`∧` - `and` и

`≡` - `==` равенство

`→` - `<=` следствие

`¬` - `(not())` отрицание
